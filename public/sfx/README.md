# 音效檔說明

請將以下音效檔放置在此目錄：

## 需要的音效檔

1. **spinning.mp3** - 抽獎轉盤音效
   - 建議：循環播放的轉盤或滾輪音效
   - 格式：MP3
   - 用途：抽獎動畫播放時的背景音效

2. **win.mp3** - 得獎音效
   - 建議：歡呼、慶祝或勝利音效
   - 格式：MP3
   - 用途：中獎結果揭曉時播放

## 注意事項

- 如果沒有音效檔，系統仍可正常運作，只是不會播放音效
- 音效檔必須是 MP3 格式
- 檔案名稱必須完全符合上述命名（區分大小寫）
- 建議音效檔大小控制在 1MB 以內，以確保載入速度
